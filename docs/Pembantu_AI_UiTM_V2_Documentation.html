1. Overview

Pembantu AI UiTM V2 is an AI chatbot for Universiti Teknologi MARA, Malaysia. It assists visitors, staff, students, and lecturers with university information in Bahasa Melayu.

Key Features

  • OpenRouter AI Integration (200+ models)
  • RAG System for knowledge base queries
  • Swedish Geometric Design (sharp edges, no rounded corners)
  • Light & Dark Theme
  • Responsive (desktop + mobile)
  • Live Reasoning Display (streaming)
  • VTube Studio Live2D lip sync
  • Voice transcription & Minimax TTS
  • WebSocket remote control

2. Installation & Setup

Prerequisites

  • Python 3.8+
  • OpenRouter API Key
  • (Optional) Minimax API Key for TTS
  • (Optional) VTube Studio + ffmpeg for avatar

Steps

git clone https://github.com/YoRzHe-HotaaRu/Pembantu-AI-UiTM-V2.git
cd Pembantu-AI-UiTM-V2
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your API keys
python app.py
App runs at http://localhost:5000

6. RAG System

Two modes: Lightweight (TF-IDF, fast, low memory) and Advanced (embeddings + ChromaDB, better accuracy, 8GB+ RAM). Set RAG_USE_ADVANCED=true for advanced mode.

Documents go in knowledge_base/ subfolders. Supports .md, .json, .pdf. Auto-indexes on startup or reload via /api/knowledge/reload.

7. VTube Studio Integration

  1. Install VTube Studio from Steam
  2. Enable API in Settings (port 8001)
  3. Set VTS_ENABLED=true in .env
  4. Restart app, toggle connection in settings

8. TTS

Minimax TTS API with Malay voice (Malay_male_1_v1). Output: MP3. Max: 10,000 chars. Optimized module streams audio in chunks.
